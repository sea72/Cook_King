# 鲫鱼豆腐汤
食材：鲫鱼，豆腐（1：1），大葱，生姜（1：2），香菜，葱花，豆油，热水，花椒，胡椒，白菜（切条）

1. 鱼洗净(开水秒烫)，去除黑膜和表层的粘膜，鱼鳞，剪去鳍
2. 吸水纸吸水，鱼改刀
3. 姜片蹭锅，油温180，煎熟煎透，翻面加姜继续煎
4. 加适量白酒(料酒)去腥，加大葱，大火炖汤（开水），捞出浮沫，20分钟
5. 放豆腐（可先行焯水去卤味），调味，加香菜

# 皮蛋瘦肉粥
食材: 姜（切丝），青菜(切丝，瓢儿白，上海青)，芝麻油，大米，糖，胡椒粉

1. 大米洗净沥干，一勺芝麻油，一小勺盐，搅拌，提前一晚急冻
2. 肉泡血，后切丝，加盐，糖，料酒，加少量水淀粉，搅拌，最后封上适量油，腌渍10分钟
3. 水烧开，加米，加10克花生油，煮2小时或高压锅上气后10分钟
4. 加入姜丝皮蛋煮五分钟，皮蛋（冻了之后可以避免流心）
5. 调味，一勺粥里加，盐，鸡粉，料酒，胡椒粉，香油，搅拌均匀撞回粥里（防止调料结块）
6. 下猪肉煮熟再煮两分钟，下葱花完成

# 油炸土豆
食材：土豆，淀粉，调料(十三香，孜然（不要太多），花椒，白芝麻，盐，辣椒)，大葱，葱花，折耳根

1. 土豆去皮，切块
2. 土豆放入开水中煮至棱角模糊（约2分钟，当土豆边角有一些煮散并且用筷子能够扎进去还有一些硬时），或者断生后捞出裹玉米淀粉？（实操中，黄皮土豆煮了很久，内部柔软度合适，外部脆度有欠缺，建议加强控水，多捏几个捏烂形成淀粉，外加玉米淀粉强化，可形成较好酥皮。）
3. 水分要沥干，可加植物油和盐2g搅拌
4. 中小火煎，不动，直到底面煎黄再翻面
5. 捞出控油后与调料拌匀

# 煎牛排

1. 把锅烧到191°以上
2. 把牛排下锅，正反面每面30秒 五成熟在锅里呆两分钟，三成熟呆1分钟
3. 如果是西冷，油边单独煎30秒
4. 烤箱设定25度，醒肉3分钟，同时插入温针
5. 如果熟度达不到，你可以把烤箱温度调整到60°，直到核心温度达到你想要的温度

# 冷煎牛排

1. 冷藏解冻，去除表面水分
2. 去筋膜，断筋，撒黑胡椒
3. 中大火每面2分钟，转小火，每面30s，再煎4-5分钟，直至侧面渗出粉红汁水，西冷侧边煎30s每次，煎2次
4.  50度核心出锅，会有3-5度的核心升温
5.  黄油，大蒜，迷迭香，百里香，黄油呈棕色，浇在牛排上
6.  切开撒盐

# 葱爆羊肉
食材：大葱2根、羊里脊肉、红辣椒半根，调料（盐、白胡椒粉、蛋清、食用油、白糖、味精、生抽、老抽、黄酒、陈醋），大蒜？

1. 羊里脊肉去皮，冲洗干净，切硬币厚薄片（羊肉可先用小苏打泡水1分钟抓洗1分钟以便更嫩）
2. 加盐、白胡椒粉、蛋清抓到黏手（粘手心不掉），再加食用油抓散（追求更嫩可以加土豆淀粉）
3. 两根大葱（炒菜时只用葱白，葱绿可以同时用来熬葱油）、半个红辣椒切滚刀块
4. 碗汁：盐1、白糖1、味精1、白胡椒粉1、生抽2、老抽1、黄酒3搅匀备用
5. 炒制，热锅凉油下里脊肉片中火煎香，再轻轻晃动，炒到全部变色，看不到生色即为八成熟，倒出备用
6. 另起锅烧油，下大葱、辣椒中火炒到微微发蔫放到羊肉上备用
7. 补油烧到冒烟，所有食材一起回锅爆香（弥补火力不足）
8. 料汁搅匀倒入，大火炒匀，后锅边烹陈醋炝香，撒孜然粉拌匀

# 酸汤肥牛
食材： 肥牛卷400g（清水浸泡洗血水），金针菇（切掉老根部，平切撕开），可选（豆芽，黄瓜，木耳），西红柿一个，生姜，大蒜，（剁碎），二荆条，小米辣（各一个），红泡椒（10g），黄灯笼辣椒酱（40g）

1. 清水加盐，煮金针菇，捞出碗中垫底
2. 肥牛下锅，变色捞出
3. 底油6层，生姜蒜末爆香，后加红泡椒，黄灯笼，再加入西红柿末，翻炒出汁水，
4. 加水开大火烧开，撇去浮沫
5. 生抽，酱油，料酒，蚝油，加肥牛中小火1分钟
6. 出锅装盘，撒辣椒圈，蒜蓉，白芝麻，泼7层热油

# 西红柿炒蛋

食材: 西红柿，大葱，小葱，蒜，可选番茄酱

1. 宽油炒蛋液（可先加点底盐），可盛油上泼，确保两面均匀熟度，后倒出
2. 番茄酱炒熟，加入大葱切段，（蒜末可选），番茄（番茄可以分两种来切，一种丁，一种块，丁可以取代番茄酱），（生抽，盐），加适量清水
3. 加入鸡蛋，后勾芡，撒上葱花

# 爆炒鱿鱼

食材：鱿鱼（会缩水，一斤只有一碗），西芹（2-3根），大葱（1根），大红椒（半个），洋葱，木耳，生姜，大蒜（1：2），干辣椒，豆瓣酱
调料：碗汁（料酒20g，生抽10g，花椒油3g，鸡精2g，胡椒粉1g，白糖少量，土豆淀粉一勺）

1. 鱿鱼去内脏，去软骨，鱿鱼中间破开，去牙齿（腿中间），去眼睛（会飙）
2. 可以考虑改刀，（切平切不断），从内切！！！而后正竖切，三刀或适量一断
3. 西芹从根处折断去撕筋，其余随便切，姜蒜切碎
4. 开水汆水鱿鱼15s，过长会老
5. 加油210度7成爆干爆香，可以适量长点时间，约45s-1min，倒出沥干
6. 加油180度6成，下葱姜蒜，干辣椒爆香，加豆瓣酱，先炒辅料配菜！！！
7. 加入鱿鱼翻炒均匀加碗汁，大火爆炒起锅

# 狼牙土豆

食材: 土豆

1. 切土豆，狼牙形，粉就少洗，脆就多洗
2. 锅烧热，宽油，5成，炸150s，粉就翻倍，预计240s
3. 捞出撒辣椒粉，而后抛6成热油，加蒜蓉辣椒
4. 食用盐2g，少量白糖，生抽5g，醋极少量，葱花，香油，搅拌装盘

# 铁板鱿鱼

1. 海鲜酱25g，芝麻酱25g，利民烧烤辣酱60g， 蚝油100g
2. 锅里加猪油35g，食用油50g
3. 油温160，撒入秦椒面，加入酱料，加200-300g水
4. 洋葱煎油180，放鱿鱼，涂上酱料，视情况加水（小的加，大的不加）
5. 撒辣椒面，孜然粉，芝麻出锅

# 辣椒炒肉

食材： 螺丝椒，青皮椒（或任意绿色的，薄皮的，注意水要沥干，宁愿细一点1*7cm长条），猪前腿肉（去除筋膜，垂直纤维切，猪颈弹，五花也可，黑猪肉最优，因脂肪风味）
进阶版：辣椒300g, 水20g，瘦肉220g，肥肉150g，酱油10g，鸡精1g，蒜3瓣，豆豉3g，盐4g，菜籽油10g

1. 肥肉切3mm厚片，瘦肉垂直切1mm薄片，瘦肉3*4，肥肉稍大，盘子热会影响风味
2. 10g底油，180度不冒烟，中火1分钟，略微起焦，加蒜，豆豉，辣椒（蒜和豆豉影响不大），30s大火，压一压
3. 起虎皮后，放入瘦肉和另一半盐，再炒一分钟，肉片变白即可，（此时可倒出多的油）
4. 沿锅边倒入酱油和鸡精，20g水，水将要收干时起锅

#  红烧排骨

食材：肋骨，葱，姜，生抽，老抽，糖 ，料酒

1. 排骨切好冷水下锅，加入葱姜，香料，花椒煮开
2. 起排骨热水冲洗
3. 起锅烧油，加入葱姜，香料，加排骨，料酒，生抽，老抽，糖，翻炒上色
4. 加热水没过排骨，加胡椒，盐，大火10-15分钟，后高压锅上汽8分钟
5. 出锅收汁

#  蒜苗回锅肉
食材：肉，红头蒜苗，生姜，大葱，花椒

1. 肉烫皮（去除皮的汗腺味）后放水里煮，加大葱，生姜，花椒，料酒，去浮沫，盖盖，小火煮15min-20min，七分熟（肉淡淡红色），泡水切薄片
2. 另切生姜，辣椒，蒜苗拍一下斜切
3. 放油炒出灯盏形，放姜和辣椒，继续放豆瓣酱，豆豉，甜面酱，酱油，可加醋
4. 放蒜头，再放蒜苗后，加盐，糖起锅


#  麻婆豆腐
食材：因卤的不同，豆腐口感不同，蛋白质含量不同，2-4%（内酯）， 6-8%(caSo4)， 8%以上(mgcl)，牛肉臊子100g（20%肥），干花椒末3g（焙），高汤（鸡精水 or 虾汤），豆瓣酱30g，豆豉10颗，大蒜5-6瓣，蒜苗，水淀粉1：6，蒜苗1根

1. 选用4%蛋白质以上豆腐，切2cm块，90度水焯豆腐1分钟，微波炉中高火2min，豆腐定型
2. 下入高汤，0.5g盐，几滴老抽上色，10min小火
3. 油，肉末，0.5g盐，黄酒，慢炒至上色炒酥，做臊子
4. 盛出一半，加豆瓣酱，豆豉，小火，上色，加蒜泥，辣椒面，略微翻炒（小心糊），后下高汤豆腐
5. 3min后勾芡多次，淀粉糊化90度，可关火，撒花椒面，蒜苗，臊子，继续勾芡
6. 出锅撒蒜苗，花椒面


# 蒜蓉空心菜

1. 空心菜一斤，大蒜50克切成颗粒
2. 锅中加清水加两勺食用盐大火烧开，放入空心菜梗焯水至断生再冷水冲凉，加3-4g食用盐、少许1-2g味精，1-2g鸡精
4. 锅中加少许底油，可加猪油，再加蒜粒炒香（约2分钟），开大火，倒入空心菜叶和菜梗翻炒30秒后空心菜断生

# 大虾烧白菜

1. 大虾剪掉须子、虾枪、沙包、剪头（别扔）、开背、去沙线、清洗干净
2. 起锅烧油，下虾头中火慢慢煸炒，按，加姜、葱，要煎透，加开水/高汤熬到浓稠，汤盛出备用
3. 另起锅，加油润锅，下白菜翻炒（干可额外加少许水）翻炒成熟倒出备用
4. 补油，煎虾，加葱姜煎透，加虾汤、白菜
5. 加盐2g, 味精1g，白糖2g，胡椒粉2g，蚝油2g搅匀继续熬煮
6. 煮到白菜软烂，大火收汁，勾芡，加明油（可以考虑芝麻油or花椒油）

#  姜爆鸭

食材：鸭，仔姜100g，生姜，辣椒（青红，小米辣，野山椒），花椒，仔姜

1. 鸭子泡水去血去腥两次，沥干后，用热宽油爆至略焦，油青亮
2. 倒出多余的油，升温，加青花椒，姜，蒜（大量），小米辣，二荆条（多），泡野山椒（黄），加豆瓣酱5g
3. 锅边加醋，加水or啤酒，加黄酒or料酒，酱油，大火去浮沫，转小火
4. 15min开始调味，盐2g，胡椒适量
5. 继续15min，转中火收汁，加味精，鸡精，白糖1g，后加仔姜，蒜叶翻炒后出锅

ps.
1. 子姜切薄，1x3比较合适
2. 辣椒要足，泡椒泡姜优先
3. 炖30分钟，时间短则肉可能咬不动

# 花甲

1. 花甲洗净外壳泥沙，置水盐质量比为1：0.032盐水浸泡，遮光2h
2. 视吐沙情况，可用开水浸泡，或者烧水焯花甲（可少加水），略微开口即可捞出，去掉一半的壳，沥干
3. 小米辣，干辣椒，葱白，青红花椒一小把，蒜蓉20g，姜一小块切丝，葱花
4. 小火爆香料头，注意不要糊，加蚝油2-4克，视口味可加适量老干妈，豆瓣酱，加花甲中火翻炒均匀
5. 料酒3g or 啤酒20g锅边加入，翻炒几下，生抽酱油4g从锅边淋入
6. 加鸡精or味精1g，少许胡椒粉，少量白糖，适量水淀粉勾芡
7. 收汁合适时，加葱花or香菜，加入少量油（芝麻 or 菜籽 or 花椒）翻炒快速起锅
